📈 MarketPulse Project Overview

MarketPulse is a Django-powered simulated trading platform that helps you practice trading, track performance, and manage risk with real-time market data.

✅ Core Features

  • 🔐 Multi-user authentication and registration
  • 📊 Live price data fetched automatically from Twelve Data API with fallback to yfinance
  • ⚡ Threshold-based alerts with email notifications and browser sound
  • 📈 Interactive candlestick and line charts with Chart.js and Plotly
  • 💾 Full CRUD operations for simulated trades
  • 💰 Automatic profit and loss calculations
  • 🛡️ Unified risk management calculator with 6 calculation modes
  • 📄 Export trades and price data as CSV and Excel
  • 🐳 Celery + Redis background tasks fetching live data every minute
  • ✅ Automated unit tests for Celery, alerts, and permissions

🧠 Key Logic & Formulas

  • 💰 Profit/Loss: (Exit Price - Entry Price) × Lot Size
  • 🛡️ Risk Amount: Account Balance × Risk %
  • 📏 RR Ratio: Reward ÷ Risk
  • ⏲️ Celery Logic: Scheduled task → Fetch API → Save model
  • 📬 Alert Trigger: If price ≥ threshold → send email + play sound
