# MarketVista Dashboard — Reviewer Walkthrough

## What this project is
MarketVista Dashboard is the monitoring and analyst-visibility layer in a broader fintech data-to-decision suite:

**DataBridge Market API → MarketVista Dashboard → RiskWise Planner → TradeIntel 360**

Its role is to help a reviewer see that this project can:
- monitor tracked assets
- surface market signals
- show data freshness
- support watchlists and alert-driven review
- route deeper investigation into adjacent products
